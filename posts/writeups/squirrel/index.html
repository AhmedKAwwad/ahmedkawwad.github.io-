it’s a bit complex but at the first look , there’s “JFIF” seems like a file extension format. after searching , it’s format for pictures but with minmal amount of data like header , let’s check the original picture hexa just the first head line to save time

└─$ xxd squirrel.jpg | head -n1                                                    
00000000: ffd8 ffe0 0010 4a46 4946 0001 0101 00b4  ......JFIF......

sounds look alike but with a little change

but is missing the file signature (in the first four bytes)

then let’s try to change this header by any hex editor in my case i used Ghex

sudo apt install ghex

and write the same header for the “.jpg” file instead the current header and now check the file it’s a picture now.

bring the writen on the top right side to Cypher Chef
And it’s encoded base32 string and BOOM The flag : flag{Ev1l_S9uirr3lz}
